# Curve Trace

Public website for Curve Trace, a desktop tool for tracing curves from charts,
log images, and scanned figures.

## Website

This repository hosts the public Curve Trace landing page with GitHub Pages.

```text
https://dancingbrushlabs.github.io/curvetrace/
```

## Local Preview

Serve the folder with any static file server:

```bash
python3 -m http.server 8082
```

Then open:

```text
http://localhost:8082
```
