Curve Trace

A desktop tool for tracing curves from modern color log documents, including PDF pages and presentation captures, then turning them into structured data.

Features

Fast, private digitizing without a complicated setup

Drop & Drag

Load an image and start working without complicated settings or setup steps.

100% Offline

Your images and digitized data stay on your computer. Nothing is sent to a server.

Quick Digitize & QC

Use the interactive interface to digitize, inspect, and quality-check results quickly.

Color-Based Well Log Digitizing

Trace color logs, with black-and-white workflows available, to produce consistent digitized results.

Color Fill Log Digitizing

Extract curve boundaries from filled color log tracks, including single-fill and two-curve fill intervals.

Modern Document Focus

Built for color logs embedded in PDFs, PowerPoint decks, reports, and screenshots rather than only legacy raster scans.

Easy Modification

Edit and refine digitized curves after extraction without restarting the workflow.

Workflow

Load, calibrate, trace, export.

  1. Import an image file or clipboard capture.
  2. Define the region of interest and axis references.
  3. Adjust curve color, fill direction, and masks to guide the trace.
  4. Review the result and export the data file.
