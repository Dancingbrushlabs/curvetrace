Curve Trace

그래프, 로그 이미지, 스캔 자료에서 곡선을 추적하고 데이터로 정리하기 위한 데스크톱 도구입니다.

다운로드 준비 중 기능 보기

Features

이미지를 데이터로 바꾸는 기본 기능

Image Calibration

축과 기준점을 지정해 이미지 좌표를 실제 데이터 범위로 맞춥니다.

Curve Tracking

색상과 마스크를 활용해 원하는 곡선을 분리하고 추적합니다.

Export

추출한 데이터를 CSV 등 분석 가능한 형식으로 정리합니다.

Workflow

불러오기, 보정, 추적, 내보내기.

  1. 이미지 또는 클립보드 자료를 불러옵니다.
  2. 관심 영역과 축 기준을 지정합니다.
  3. 곡선 색상과 마스크를 조정해 추적합니다.
  4. 결과를 검토하고 데이터 파일로 내보냅니다.
